Chashitsu Common CSS (CCCSS)
====================

Chashitsu Common CSS (CCCSS) は、複雑なレイアウトを持たないHTMLページを低コストで整形するスタイルシート（予定地）です。

## ビルド
node/npm が必要です。
``` sh
$ npm i
$ npm run build
```
で `dist/` 下にCSSが生成されます。

## サンプルを動作させる
node/npm が必要です。
``` sh
$ npm i
$ npm start
```
でブラウザが立ち上がります。
